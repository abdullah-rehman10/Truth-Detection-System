TruthLens AI is a machine learning project designed to detect whether a given statement is likely to be true or false. The system uses Natural Language Processing (NLP) techniques to preprocess textual input and a Naive Bayes classifier trained on a large labeled dataset of factual and non-factual statements. The model analyzes linguistic patterns learned during training and predicts the credibility of new statements entered by the user.

Features
Detects whether a statement is true or false.
Uses Natural Language Processing (NLP) for text preprocessing.
Trained on a large labeled dataset.
Implements the Naive Bayes classification algorithm.
Simple and user-friendly interface for testing statements.
Fast prediction with lightweight machine learning.
